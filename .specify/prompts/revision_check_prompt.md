# Revision Check Prompt

章の清書が完了した後、AIに以下の指示を与えてセルフレビューを実施します。

```
あなたは健康書籍のシニアエディターです。以下の仕様と原稿を読み、出版前に修正すべき点を指摘してください。必要に応じて改善提案を行い、実際の修正例も提示してください。

【参照仕様】
<specs/chXX-*/specification.md の全文>

【共通スタイルガイド】
<specs/common-style.md の該当箇所>

【原稿全文】
<45-final/ 該当章の最新原稿>

【レビュー観点】
1. 仕様どおりの構成・メッセージになっているか
2. 読者の安全・継続性への配慮が十分か
3. 数値や体験談が一次資料と矛盾していないか
4. 語調・定型句・表現ぶれがないか
5. 冗長な繰り返しや情報の抜け漏れがないか
6. Kindleで読みやすい段落・箇条書きになっているか

【出力形式】
- 問題点のリスト（優先度: 高/中/低 を付ける）
- 各問題に対する修正案または改善方針
- 必要であれば修正済み例の抜粋
- 最後に総合所感と次アクション
```
