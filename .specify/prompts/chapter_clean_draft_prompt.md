# Chapter Clean Draft Prompt

以下の手順で章の清書を生成します。AIに渡す際は必要箇所を埋めて使用してください。

```
あなたは健康書籍のプロライターです。以下の仕様・資料を厳守して、Kindle出版レベルの清書原稿を作成してください。

【対象章】
- 章番号とタイトル: <例: 第1章 ころも式の約束と前提>

【参照仕様】
<specs/chXX-*/specification.md の全文を貼り付け>

【共通スタイルガイド】
<specs/common-style.md の関連部分を必要に応じて貼る>

【参照ドラフト】
<40-drafts/ 該当ファイルの引用したい部分のみ抜粋>

【参照セッション / メモ】
<25-sessions/ と 20-memos/ から必要部分を抜粋>

【出力要件】
- Markdown形式（見出し、箇条書き、段落すべて整形済み）
- 章冒頭は読者の悩みと約束を提示する導入から開始
- 章末にまとめ・行動リスト・次章への橋渡しを含める
- 仕様書で指定したキーメッセージ・注意喚起・数値を必ず反映
- 仕様にない新情報を勝手に追加しない
- 段落は200字前後を目安に区切る

【確認事項】
- 「無理はしないでください」「人それぞれですが」などの定型句を適所に配置
- 医学的判断が必要な部分では専門医への相談を促す
- 成功談と失敗談をバランスよく盛り込み、共感と安心感を与える
- 4つのペルソナ全員への配慮を明示的に行う

以上を踏まえて、完成原稿のみを出力してください。解説や注釈は不要です。
```
